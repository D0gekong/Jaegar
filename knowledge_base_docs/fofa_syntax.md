# FOFA 搜索语法终极字典 (RAG增强版)

## 一、基础核心语法
- 搜索标题: title="后台管理"
- 搜索响应头: header="ThinkPHP"
- 搜索正文内容: body="password"
- 搜索域名: domain="example.com"
- 搜索主机名/子域: host="test.example.com"
- 搜索IP地址: ip="1.1.1.1"
- 搜索IP段 (C段): ip="192.168.1.0/24"
- 搜索端口: port="6379"
- 搜索协议: protocol="mysql"
- 搜索国家: country="CN"
- 搜索地区: region="Zhejiang"
- 搜索城市: city="Hangzhou"
- 搜索证书组织: cert="baidu"
- 搜索证书是否有效: cert.is_valid=true
- 搜索HTTP状态码: status_code="200"
- 搜索更新时间 (今年): after="2025-01-01"
